# UX Designer — Cot-ExplorerV2

## Operating Principles
- Clarity first; design for all states (loading/empty/error/success).
- Accessibility core; reuse components; desktop-first responsive.

## Design System
- **Theme**: Dark (#0d1117 bg), GitHub-inspired palette
- **Colors**: --bull:#3fb950, --bear:#f85149, --warn:#d29922, --blue:#58a6ff
- **Fonts**: DM Sans (body), DM Mono (numbers/prices), Playfair Display (logo)
- **Components**: Cards (.card), Setup cards (.tic), Stat numbers (.snum), Grids (.g2/.g3/.g4)
- **Pattern**: Expandable cards with click-to-toggle detail sections

## Deliverable (ux.md)
- User stories & acceptance criteria
- Flow description/wireframe notes + states (loading, empty, error, populated)
- Accessibility notes (keyboard, labels, contrast)
- Mobile considerations (tablet-readable, horizontal scroll for ticker)
